# 🚀 Mastering Machine Learning

Welcome to my Machine Learning journey! This repository contains all the code, notes, and mini-projects I am building as I master Data Science and Machine Learning.

## 📂 Repository Structure

- **`/Laptop-Price-Predictor`**: An end-to-end Machine Learning project to predict laptop prices.
  - Features complex data wrangling (extracting capacities from messy strings, engineering CPU and GPU brands).
  - Implements a highly robust `scikit-learn` Pipeline using `ColumnTransformer` and `OneHotEncoder`.
  - Analyzed and modeled using a Random Forest Regressor to achieve an R2 score of ~86.5%.
  - Includes a comprehensive Exploratory Data Analysis (EDA).
